# UI Guide

## Design Direction

UI ต้องสะอาด อ่านง่าย และเหมาะกับผู้เรียนไทยที่เริ่มฝึกภาษาอังกฤษ

เป้าหมายคือทำให้ผู้เรียนรู้สึกว่าเปิดแอปแล้วเรียนต่อได้ทันที ไม่ต้องตีความหน้าจอเยอะ

## Tone

- ภาษาไทยเป็นหลัก
- ภาษาอังกฤษเป็นรอง แต่ต้องเด่นพอสำหรับคำศัพท์
- น้ำเสียงเป็นมิตร ชัดเจน ไม่กดดัน
- ไม่ทำให้ดูเหมือนเกมเด็กเกินไป
- เหมาะกับนักเรียน นักศึกษา และคนเริ่มทำงาน

## Layout

- Mobile-first
- ใช้ Card layout สำหรับคำศัพท์และโหมดฝึก
- ระยะห่างเพียงพอสำหรับการแตะบนมือถือ
- หน้าแรกควรเห็นความคืบหน้าและปุ่มเริ่มเรียนได้เร็ว
- Desktop ใช้พื้นที่เพิ่มเพื่อแสดงรายการและรายละเอียดได้สบายขึ้น

## Visual Style

- พื้นหลังอ่อน
- โทนสีสะอาด น่าเรียน
- ใช้สี accent สำหรับปุ่มหลักและสถานะสำคัญ
- หลีกเลี่ยงสีจัดเกินไป
- ใช้เงาและเส้นขอบอย่างพอดี

## Buttons

- ปุ่มใหญ่ ใช้ง่ายบนมือถือ
- ปุ่มหลักควรชัดเจน เช่น เริ่มฝึก, ดูคำตอบ, ตอบถูก, ตอบผิด
- ปุ่มรองควรไม่แย่งความสนใจจากปุ่มหลัก
- มีสถานะ disabled เมื่อกดยังไม่ได้

## Typography

- ฟอนต์อ่านง่าย
- คำศัพท์อังกฤษควรใหญ่กว่าคำอธิบาย
- คำแปลไทยต้องอ่านง่ายมาก
- IPA และคำอ่านง่ายควรอยู่ใกล้คำศัพท์
- หลีกเลี่ยงข้อความยาวใน card เดียว

## Components

ส่วนประกอบหลักที่ควรมี:

- App header
- Progress summary
- Vocabulary card
- Word detail card
- Flashcard
- Quiz option button
- Review action buttons
- Search input
- Filter controls
- Status badge

## Accessibility

- ขนาดตัวอักษรต้องอ่านได้บนมือถือ
- ปุ่มต้องแตะง่าย
- contrast เพียงพอ
- หลีกเลี่ยงการใช้สีเพียงอย่างเดียวในการสื่อสถานะ
- รองรับ keyboard navigation ในส่วนหลักเมื่อทำได้
