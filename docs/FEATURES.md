# Features

## Vocabulary List

แสดงรายการคำศัพท์ทั้งหมดในชุดเรียน ผู้เรียนเห็นคำศัพท์ คำแปลไทย ระดับ CEFR และสถานะการเรียนได้อย่างรวดเร็ว

## Word Detail

แสดงรายละเอียดของคำศัพท์แต่ละคำ:

- คำศัพท์ภาษาอังกฤษ
- IPA
- คำอ่านง่ายสำหรับผู้เรียนไทย
- ชนิดของคำ
- คำแปลไทย
- ความหมายภาษาอังกฤษแบบง่าย
- ตัวอย่างประโยค
- บริบทการใช้ 3 แบบ: daily, work, study
- synonyms
- common mistake
- memory tip

## Flashcard Mode

โหมดฝึกจำแบบการ์ด ผู้เรียนดูคำศัพท์ก่อน แล้วกดเพื่อดูคำตอบ รายละเอียด และตัวอย่าง

เป้าหมายคือช่วยให้ผู้เรียนจำคำศัพท์โดยไม่รู้สึกหนักเกินไป

## Quiz Mode

โหมดทดสอบคำศัพท์แบบง่าย เช่น เลือกคำแปลที่ถูกต้อง หรือเลือกคำศัพท์จากความหมายภาษาไทย

ใน MVP จะเน้นคำถามที่ตรวจคำตอบได้ชัดเจนและไม่ต้องใช้ backend

## Review Mode

โหมดทบทวนคำที่อยู่ในสถานะ Learning หรือ Review โดยเรียงจากคำที่ควรฝึกก่อน

ผู้เรียนสามารถกดว่าตอบถูกหรือตอบผิดเพื่ออัปเดตสถานะคำศัพท์

## Progress Dashboard

แสดงภาพรวมความคืบหน้า:

- จำนวนคำทั้งหมด
- จำนวนคำใหม่
- จำนวนคำที่กำลังเรียน
- จำนวนคำที่ต้องทบทวน
- จำนวนคำที่จำได้แล้ว

## LocalStorage Progress

เก็บสถานะคำศัพท์ใน browser ด้วย LocalStorage โดยไม่ต้องใช้ backend หรือ database

ข้อมูลที่เก็บ:

- สถานะของแต่ละคำ
- จำนวนครั้งที่ตอบถูก
- จำนวนครั้งที่ตอบผิด
- วันที่ทบทวนล่าสุด

## Search / Filter

ค้นหาและกรองคำศัพท์เบื้องต้น:

- ค้นหาจากคำศัพท์อังกฤษ
- ค้นหาจากคำแปลไทย
- กรองตาม CEFR
- กรองตามสถานะการเรียน
- กรองตามชนิดของคำ
