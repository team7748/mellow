# Project Brief

## Problem

ผู้เริ่มต้นชาวไทยจำนวนมากจำคำศัพท์ภาษาอังกฤษได้ยาก เพราะเรียนคำศัพท์แบบแยกคำ ไม่เห็นบริบทการใช้จริง และไม่มีระบบทบทวนที่ช่วยให้กลับมาฝึกคำที่ยังไม่แม่น

เว็บนี้แก้ปัญหาโดยทำให้ผู้เรียนเห็นคำศัพท์พร้อมคำแปลไทย คำอ่าน ความหมายแบบง่าย ตัวอย่างประโยค และบริบทการใช้หลายแบบในหน้าจอที่ใช้งานง่ายบนมือถือ

## Target Audience

- นักเรียนไทยที่เริ่มฝึกคำศัพท์อังกฤษ
- นักศึกษาที่ต้องการเพิ่มคลังคำพื้นฐาน
- คนเริ่มทำงานที่อยากใช้คำศัพท์ทั่วไปในชีวิตประจำวันและที่ทำงาน
- ผู้เรียนระดับเริ่มต้นถึงกลางต้น โดยเริ่มจาก CEFR A1-A2

## Main Goals

- ทำให้ผู้เรียนจำคำศัพท์เป็นชุดเล็ก ๆ ได้ง่าย
- แสดงคำแปลไทย IPA คำอ่านง่าย และตัวอย่างที่เข้าใจเร็ว
- ให้ผู้เรียนฝึกซ้ำผ่าน Flashcard, Quiz และ Review
- เก็บความคืบหน้าใน LocalStorage โดยไม่ต้องสมัครสมาชิก
- วางโครงสร้างข้อมูลให้ขยายจาก 60 คำเป็น 1,000 คำได้ง่าย

## MVP Scope

- ข้อมูลคำศัพท์เริ่มต้น 60 คำ
- Vocabulary List สำหรับดูรายการคำศัพท์
- Word Detail สำหรับดูข้อมูลคำศัพท์แบบละเอียด
- Flashcard Mode สำหรับฝึกจำ
- Quiz Mode สำหรับทดสอบ
- Review Mode สำหรับทบทวนคำตามสถานะ
- Progress Dashboard แบบเรียบง่าย
- Search และ Filter เบื้องต้น
- บันทึกสถานะด้วย LocalStorage
- UI แบบ mobile-first

## Not Included In Version 1

- Backend
- Database
- ระบบสมัครสมาชิกหรือ Login
- ระบบเสียงอ่านคำศัพท์
- AI สร้างคำศัพท์หรือประโยค
- ระบบจัดคอร์สซับซ้อน
- Leaderboard หรือ gamification หนัก ๆ
- Sync ข้อมูลข้ามอุปกรณ์
- Admin panel สำหรับจัดการคำศัพท์
